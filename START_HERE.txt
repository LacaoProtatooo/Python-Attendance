## âš ï¸ Common Issues & Solutions

### Problem: "Student not found in database"
**Solution:**
1. Run: `python migrate_students.py`
2. This adds all existing students from images to database
3. Try attendance again

### Problem: Face not detected
**Solution:**
- âœ“ Better lighting (face clearly visible)
- âœ“ Move closer to camera (30-60 cm)
- âœ“ Remove sunglasses/masks
- âœ“ Keep face in frame

### Problem: "No trained model found"
**Solution:**
- Register at least 1 student first
- Model auto-created after registration

### Problem: "Already marked today"
**Solution:**
- Expected! Each student marks once per day
- Try next day to mark again
- Different students can mark same day

### Problem: Camera not working
**Solution:**
- Close Zoom/Teams (they use camera)
- Check Windows camera permissions
- Restart application
- Restart computer if needed

### Problem: Button won't click or GUI freezes
**Solution:**
- Kill and restart application
- Make sure previous window is fully closed
- Check error messages in terminal
